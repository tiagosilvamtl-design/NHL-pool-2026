OVERVIEW
--------
A free, automated NHL playoffs predictions pool hosted on GitHub Pages.
Friends submit picks via a web form (no login required). Picks lock
automatically when each series starts. Scores update hourly via the NHL
public API. No ongoing cost — everything runs on free tiers.

Live URL: https://tiagosilvamtl-design.github.io/NHL-pool-2026/


--------------------------------------------------------------------------------
SCORING RULES
--------------------------------------------------------------------------------
  +2 pts  Correct series winner
  +1 pt   Correct number of games (4/5/6/7)
          — only awarded if the winner pick is also correct

--------------------------------------------------------------------------------
KEY BEHAVIOURS
--------------------------------------------------------------------------------

  Submission
    - Anyone with the link can submit picks — no login required
    - Identified by email address (unique per person)
    - Picks can be updated until the series locks
    - Each series locks independently when its first game starts
    - On re-submission with same email, existing picks are overwritten

  Locking
    - Lock time is determined from the NHL schedule API (first game UTC time)
    - Lock status is computed by GitHub Actions hourly and stored in the Sheet
    - Locked series show a "LOCKED" badge; inputs are disabled on the form

  Scoring (computed live by Apps Script on every leaderboard load)
    - Scans all completed series against all submissions
    - +2 for correct winner, +1 bonus for correct games (winner must be correct)
    - Leaderboard sorted by total, ties share rank

  Automation
    - GitHub Actions runs every hour
    - Fetches NHL carousel API for series results and team logos
    - Fetches NHL schedule API to determine first game times
    - POSTs results to Apps Script with secret token authentication
    - Apps Script upserts the series sheet

  Easter egg
    - If a participant picks against the Montreal Canadiens, a full-screen
      "SHAME" animation fires after submission (falling text, bell ringer)
